The COLORS web application allows a user to manage an array of colors. The user can add colors to his list of colors and search the list for a particular color.



The web application requires a user to log in prior to utilizing any features.



**The technologies used for production:**

* Digital Ocean (Web Hosting)
* MySQL (Database)
* Git Bash (Local Version Control)
* GitHub (Collaboration Version Control)
* WebStorm IDE for Html and JavaScript scripting
* PHPStorm IDE for PHP scripting
* Putty for file upload to server





**Set-up Instructions:**

* Set up hosting with Digital Ocean and acquire a domain
* Create a LAMP Droplet
* Set up passwords and user accounts for your server
* In the DNS manager on your server, have it point to your domain
* Set up MySQL database and all tables needed
* Set up LAMPAPI and produce API through PHP scripts
* Set up front end with html and JavaScript scripts
* Upload all files to server
* Upload all image files needed to server



**Accessing the application:**

* Load favorite internet browser
* In the URL field type: http://calaren.net and then hit ENTER key
* Once the web application loads, enter your login credentials in the respective fields
* Once you are logged in, you will be able to utilize the application features



**Limitations and Assumptions:**



The web application does not have a user registration feature. All users must be preemptively added to the database prior to being able to utilize the web application.



It is assumed that the user has already included into the table of users.
